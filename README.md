# soosho.dev

Personal website and portfolio — built with Next.js.

## About

This is my corner of the web. Built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Font:** Geist (by Vercel)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Deployment

Deployed via [Vercel](https://vercel.com).
